# JChat Pro

JChat Pro is the advanced evolution of the JChat series, transitioning from a simple chat interface to a feature-rich, community-driven communication platform. It combines modern web technologies with low-latency audio protocols to deliver a seamless user experience.

---

## 🛠️ Tech Stack

* **Python**: Core programming language for backend logic and server management.
* **HTML/CSS/JS**: Primary languages for the Graphical User Interface (GUI), featuring a Discord-inspired dark theme.
* **Flask-SocketIO**: Handles real-time bi-directional messaging and high-capacity file handling.
* **PyAudio**: Powering audio capturing and playback for integrated communication.
* **UDP Protocol**: Utilized for low-latency voice transmission to ensure real-time audio performance.
* **JSON**: Lightweight data storage for user accounts, profiles, and persistent chat history.

## 🌟 Key Features (Pro Version)

* **Modern Discord UI**: A fully responsive, high-fidelity replica of the Discord interface.
* **Media Sharing**: Support for file, image, and audio clip uploads (up to 20MB).
* **Voice Integration**: Peer-to-peer voice signaling and UDP-backed audio transmission.
* **Data Persistence**: All messages and user data are securely saved and loaded via a localized JSON database.
* **Server Customization**: Support for custom server emojis and personalized user profiles.

## 📦 Quick Start

1.  **Install Dependencies**:
    ```bash
    pip install flask flask-socketio
    ```
2.  **Run the Server**:
    ```bash
    python server.py
    ```
3.  **Access the App**:
    Open `http://localhost:5000` or use the vpn and `http://vpn-ip:5000` in your web browser.
