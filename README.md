# JChat Pro - Advanced Real-Time Communication

JChat Pro is the professional evolution of the JChat series. It transitions from a simple chat interface to a feature-rich community platform. Built with **Flask** and **Socket.io**, it offers a seamless, high-performance experience with a modern dark-themed UI.

## 🌟 What's New in Pro?

Compared to the [Basic Version](https://github.com/Jjoweek0819/Jchat-Basic-Ver-Beta0.1.1), the Pro version introduces:

- **WebRTC Voice Signaling**: Peer-to-peer voice communication support.
- **Rich Media Handling**: Integrated file upload system (up to 20MB) for images, audio, and documents.
- **Custom Emoji System**: Upload and use server-specific emojis.
- **Enhanced UI/UX**: Improved Discord-inspired layout with message grouping and typing indicators.
- **Persistent Storage**: Full account and message history persistence using localized JSON data.

## 🛠️ Key Features

- **Instant Messaging**: Low-latency communication via WebSockets.
- **Dynamic Rooms**: Create and join different channels with real-time updates.
- **User Profiles**: Custom avatars and bio management saved server-side.
- **Safety & Stability**: Threaded async mode and large buffer handling for media files.
- **Responsive Design**: Optimized for various screen sizes with CSS Grid and Flexbox.
